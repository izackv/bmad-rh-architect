# Red Hat Platform Readiness Checklist

- [ ] Target Red Hat products and versions are documented.
- [ ] Support model (subscription / partner / internal) is identified.
- [ ] Lifecycle and upgrade strategy is defined.
- [ ] Security baselines and hardening guidelines are referenced.
- [ ] Backup and DR strategy is documented.
- [ ] Monitoring and alerting requirements are defined.
- [ ] Capacity and scaling strategy is described.
- [ ] Operational ownership (who runs what) is clear.
