# Task: rh-create-project-plan

## Purpose

Create a structured project plan that can be imported into a PM tool
or used as a detailed implementation guide.

## Steps

1. Confirm:
   - Approved scope (from HLD/LLD).
   - Any fixed deadlines (regulatory/contractual).

2. Load structure from `templates/rh-project-plan-template.yaml`.

3. Build:
   - WBS (work breakdown structure).
   - Milestones and deliverables.
   - Dependencies and risk register.
   - Governance and communication plan.

4. Align with timeline and cost estimates:
   - Cross-reference major milestones and cost peaks.

5. Produce:
   - Plan in a readable markdown/table form.
   - Optional: a list of tasks suitable for import into common PM tools.
